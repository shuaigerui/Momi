# MotorcycleSocialRp（机车社交）开发文档

## 产品定位
机车骑行社交 App：组队骑行、短视频分享、俱乐部、即时消息与视频通话、钱包金币消耗。

## Tab 结构（4 Tab）
1. **Home / Team Group** — 组队骑行列表
2. **Video** — 全屏短视频流
3. **Club** — 俱乐部双列网格
4. **Profile** — 个人中心

## 账户与会话
- 当前用户单例：`AccountSession.shared`
- 预设体验账号：`test@gmail.com` / `123456`（登录后有完整社交痕迹）
- 内置 3 名骑手用户（各 1 条视频动态 + 1～4 条评论 + 话题）
- 首页另有 6 名组队发布者（每人仅 1 张图片动态，个人主页展示）
- Apple 登录 / 邮箱新注册：新用户态（无会话、无关注粉丝数、余额 0）
- 非预设邮箱登录：未注册提示并跳转注册；已注册本地持久化
- 登录态持久化，下次启动直达首页

## 业务规则（实现必守）
1. **互关才能 Chat**：未互关时 Chat 按钮浅色切图 + 弹窗提示
2. **金币**：发布动态 / 加入组队各消耗 **10** 金币；余额不足弹窗引导充值；发布入口带锁暗示
3. **余额**：所有账号初始余额 0，内购到账本地缓存刷新
4. **举报/拉黑/不感兴趣**：真实写入本地，黑名单在设置可见
5. **全局弹窗**：禁止 `UIAlertController`，使用主题渐变通用弹窗
6. **骨架屏**：主要列表/主页首次进入需骨架 + loading
7. **权限**：相机/相册/麦克风文案写入 Info.plist；拒绝后引导设置
8. **短屏适配**：内容区可滚动；标题/返回/更多/设置固定不随滚动
9. **协议**：隐私/用户协议 WebView，URL 可配置占位
10. **注销/退出**：二次确认后回登录页

### 网络层（拟真）
- 主域名：`opi.9gnprx32.link`（见配置文件）
- AES-128 CBC：`aes_key` / `aes_iv` 来自配置
- 请求体加密、响应解密；子路径按事件命名（如 `/v1/auth/signin`）
- 本地业务仍落盘，网络层模拟延迟与加解密管线

### 内购（StoreKit 真实）
商品 ID 与钻石数见 `FL-Momxz.txt`，测试/正式统一走真实 StoreKit。

### 审核相关约定（仅文档，禁止写入代码注释/字符串）
